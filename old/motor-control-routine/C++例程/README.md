## 概述
- 这里是 C++ 例程总入口，按适配器类型分为 `u2can` 和 `u2canfd` 两条路线。
- 本页只负责帮你选路；具体安装、编译和运行流程分别在子目录自己的 `USAGE.md` 或 `WORKFLOW.md` 中。

## 文档 / 资源
- [u2can/README.md](u2can/README.md) - USB 转 CAN 路线；详细流程在 `USAGE.md`
- [u2canfd/README.md](u2canfd/README.md) - USB 转 CANFD 路线；详细流程在 `WORKFLOW.md`
- [README.en.md](README.en.md) - 英文轻量入口

## 快速开始
- 如果你使用 USB 转 CAN，先看 [u2can/README.md](u2can/README.md)。
- 如果你使用 USB 转 CANFD，先看 [u2canfd/README.md](u2canfd/README.md)。
